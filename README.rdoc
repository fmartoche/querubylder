== Querubylder

The objective of querubylder is to provide an easy way to query data from various sources (several tables). People just say what data they want as an output (the columns they want in their spreadsheet) and querubylder builds the appropriate SQL query.

You can find a demo {here}[http://querubylder.herokuapp.com] ({http://querubylder.herokuapp.com}[http://querubylder.herokuapp.com]).
